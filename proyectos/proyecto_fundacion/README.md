# 🌱 Fundación Sembrando Futuro — Sitio Web Oficial

> *"Sembrando Código, Cosechando Futuro"*  
> Sitio web institucional para una fundación que enseña programación **gratuita** a jóvenes de bajos recursos en Bucaramanga y municipios vecinos de Santander, Colombia.

---

## 📋 Descripción del Proyecto

Este proyecto fue desarrollado como ejercicio académico de diseño y desarrollo frontend para el cliente ficticio **Luis Hernández**, director de la Fundación Sembrando Futuro. El objetivo: proyectar una imagen seria, transparente y profesional que rompa el estigma de desorganización en las fundaciones, y que ayude a conseguir donantes y voluntarios.

**Institución:** ADSO — Ficha 3174193  
**Fecha de inicio:** 08/05/2026  
**Autores:** Juan David Castro Rojas · Daniel Yaced Suárez Ariza · Sebastián Morales

---

## 🎯 Objetivos del Sitio

- Comunicar el impacto real de la fundación con datos verificables
- Facilitar el proceso de donación (3 métodos disponibles)
- Reclutar voluntarios profesionales del sector tech
- Transmitir confianza y empoderamento — no lástima
- Demostrar transparencia financiera con informes públicos

## 🖥️ Secciones de la Página

| # | Sección | Descripción |
|---|---------|-------------|
| 1 | **Nav** | Navegación fija con logo, links y CTA "Donar" |
| 2 | **Hero** | Misión clara + dos CTAs principales |
| 3 | **Impacto** | 4 estadísticas en grid: 250+ jóvenes, 18 municipios, 85%, 5 años |
| 4 | **Programas** | Frontend Junior, Backend Junior, Diseño UX — todos gratuitos |
| 5 | **Historias de Éxito** | 3 egresados con testimonio, foto y empresa actual |
| 6 | **Cómo Donar** | Transferencia bancaria, PSE/Nequi/Daviplata, Padrinazgo mensual |
| 7 | **Ser Voluntario** | Formulario: nombre, profesión, horas disponibles, área de interés |
| 8 | **Aliados** | Logos de empresas y organizaciones aliadas |
| 9 | **Newsletter + Transparencia** | Suscripción + links a informes financieros auditados |
| 10 | **Footer** | Contacto completo, redes sociales, links de transparencia |

## 🎨 Sistema de Diseño

### Paleta de Colores

| Nombre | Hex | Uso |
|--------|-----|-----|
| Azul Confianza | `#07172A` | Header, fondos principales, botones |
| Verde Esperanza | `#22C55E` | Acentos, CTAs, iconos, bordes activos |
| Blanco | `#FFFFFF` | Fondo principal, tarjetas |
| Gris Neutro | `#E5E7EB` | Tarjetas secundarias, separadores |

### Tipografía

| Uso | Fuente | Peso |
|-----|--------|------|
| Títulos y headings | **Poppins** | Bold (700 / 800) |
| Cuerpo y párrafos | **Open Sans** | Regular (400 / 500) |

> Ambas fuentes son modernas, limpias y se ven profesionales juntas. Mantienen buena legibilidad en pantalla.

### Personalidad de Marca

- **Tono:** Formal, técnico y transparente — cercano a los jóvenes, atractivo para grandes donantes
- **Valores:** Transparencia · Excelencia · Equidad
- **Lo que se evita:** Sentimiento de lástima o victimización; el sitio transmite **empoderamiento**

---

## ⚙️ Tecnologías Utilizadas

- **HTML5** semántico
- **CSS3** — Variables CSS, Flexbox, CSS Grid, animaciones
- **JavaScript** vanilla — Intersection Observer para animaciones al scroll
- **Google Fonts** — Poppins + Open Sans
- Sin frameworks, sin dependencias externas — carga rápida y fácil de mantener

## 🚀 Cómo Usar

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/sembrando-futuro.git
   ```

2. Abre el archivo directamente en tu navegador:
   ```bash
   open index.html
   # o simplemente haz doble clic en el archivo
   ```

3. No requiere servidor local ni instalación de dependencias.
